# Jupyterhub Docker Modus

Anders als bei der Installation und Konfiguration auf einer VM sind bei Docker alle Installationen und Konfigurationen bereits vorbereitet. 
Dank der Tatsache,dass Docker nicht vom Operationssystem abhängig ist, müssen die Benutzer nur den Docker-Container einsetzen.

2. [Deployment von Jupyterhub Docker Modus](/Jupyter_Docker/Deploy) gibt eine Schritt für Schritt Anleitung für Deployment Jupyterhub Docker Modus.
3. Ein [Authentifizierungssystem](/Jupyter_Docker/Shibboleth) mit Shibboleth für Jupyterhub.
4. [Konfiguration von Jupyterhub](/Jupyter_Docker/Hub)
