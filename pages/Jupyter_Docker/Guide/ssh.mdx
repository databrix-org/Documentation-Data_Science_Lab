# Einloggen in Manager Node

Die Programmierumgebung für Ihren Kurs wird in der bwCloud bereitgestellt. 
Genauer gesagt, wird sie in virtuellen Machinen (VM) in bwCloud deployed. 
Sie müssen sich zunächst per Remote-Zugang in eine bestimmte VM einloggen, um Ihre Programmierumgebung zu verwalten. 
Diese VM wird als Manager Node bezeichnet.
Er verwaltet alle anderen Worker Nodes, also die VMs, die Rechenaufgaben durchführt.

Führen Sie folgende Schritte aus, um sich in Manager Node einzuloggen:

Voraussetzung:
Sie müssen Ihren eigenen privaten SSH Schlüssel mit diesem [Public Schlüssel generieren](https://owncloud.dhbw-stuttgart.de/index.php/apps/files?dir=/Databrix/bwCloud-Credentials&fileid=27525054).
Sie müssen diesen Public Schlüssel herunterladen und anhand dieses Schlüssel Ihren eigenen privaten Schlüssel generieren.
Für generation privaten SSH Schlüssel sehen Sie die [Anleitung](https://www.bw-cloud.org/en/bwcloud_scope/use) von bwCloud 

1. Öffnen Sie einen Terminal.
2. Öffnen Sie den Odner in Terminal, wo ihrer privater SSH Schlüssel gespeichert ist.
3. Führen Sie folgenden Befehl aus, um mit ihrer VM zu verbinden
``` js copy
ssh -i [privater SSH Schlüssel] ubuntu@[Hostname]
```
Sie müssen [privater SSH Schlüssel] und [Hostname] ihrer VM mit echten Daten ersetzen. 
4. Geben Sie den Passwort ein.
