# Deployment in bwCloud

Wie in der Einleitung beschrieben, sollte für jede VM eine eigene JD eingesetzt werden. Je nach Anzahl der Teilnehmer und Gruppen sollte der Kursanbieter die Anzahl der Gruppen festlegen. Für einen Kurs mit dreißig Nutzern, die in drei Gruppen aufgeteilt sind, wären beispielsweise zehn VMs erforderlich. Je nach den für die Aufgabe erforderlichen Rechenressourcen sollte eine VM mit vier CPUs und 32 GB RAM ausgewählt werden. Die JD sollte dann auf der VM aus dem Docker-Image bereitgestellt werden. Der Einsatz ist ein zweistufiger Prozess:
1.	SSL Zertifikat erstellen
2.	Docker-compose.yml laufen
